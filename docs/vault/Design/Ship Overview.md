# Ship Overview

## ISV Prometheus

A generation ship carrying 5,000–10,000 crew and passengers on a deep space voyage.

### Key Stats
| Parameter | Value |
|-----------|-------|
| Population | ~7,500 (5,000 crew + 2,500 passengers) |
| Ship length | ~400m |
| Ship beam (width) | ~65m amidships |
| Habitable decks | ~20 |
| Deck height | 3m habitable + 1m interstitial |
| Private space | 15–25 m²/person |
| Total habitable area | ~300,000 m² |

### Hull Shape
Tapered — narrow at bow (command) and stern (engineering), widest amidships (habitation).

| Decks | Width | Length |
|-------|-------|--------|
| 0–1 (Command) | 40m | 200m |
| 2 to N-3 (Mid) | 65m | 400m |
| Last 2 (Engineering) | 50m | 300m |

## Deck Zoning Plan

```
Deck  1:  Bridge, Command, Communications, Captain's quarters
Deck  2:  Administration, Security, Conference, Brig
Deck  3:  VIP suites, Officer quarters, Wardrooms
Decks 4–5: Crew habitation (single cabins, shared baths, laundry)
    [Service deck A]
Decks 6–7: Crew habitation + scattered cafés/lounges
Deck  8:  Passenger habitation (family suites)
    [Service deck B]
Deck  9:  Passenger habitation + recreation
Deck 10:  Main mess halls, galleys, food storage, bakery
Deck 11:  Medical — hospital, surgery, dental, pharmacy, quarantine
    [Service deck C]
Deck 12:  Recreation — gym, theatre, library, game rooms, bars
Deck 13:  Recreation — arboretum/park, pool, observation lounges
Deck 14:  Education — schools, classrooms, nursery, training sims
    [Service deck D]
Deck 15:  Hydroponics farms (upper)
Deck 16:  Hydroponics farms (lower) + atmosphere processing
Deck 17:  Water recycling, waste processing, environmental monitoring
    [Service deck E]
Deck 18:  Cargo bays, general storage, armory
Deck 19:  Shuttle bay, EVA airlocks, docking
Deck 20:  Main reactor, backup reactor, fuel storage
Deck 21:  Main engines, machine shops, electronics labs, robotics
```
